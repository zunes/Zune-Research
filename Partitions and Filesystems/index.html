    Partitions and Filesystems¶

    As the Zune is built on top of the NVIDIA Tegra APX 2600 platform it might use something similar to the default partition layout.

    Partition Layout from the Registry¶

    HKEY_LOCAL_MACHINE\System\StorageManager\PartitionTable shows the partition layout of the Zune.

    "01"="FATFS"
"04"="FATFS"
"06"="FATFS"
"07"="MSIFS"
"0B"="FATFS"
"0C"="FATFS"
"0E"="FATFS"
"0F"="FATFS"
"20"="BOOT"
"21"="ZBINFS"
"22"="RAWFS"
"23"="RAWFS"
"25"="IMGFS"
"26"="BINARY"

    Filesystems¶

    ZCSTFS¶

    Zune Content File System DLL: zcstfs.dll This is the Filesystem for the XNA Storage containers stored as ZCP Files.

    ZBINFS¶

    Zune Binary File System DLL: zbinfs.dll

    FATFS¶

    FAT File System DLL: zexfat.dll

    EXFAT¶

    DLL: zexfat.dll exFAT File System

    MSIFS¶

    MSIFS uses the EXFAT format.

    ObjectStore¶

    DLL: filesys.dll
